## Overview

HealthTrack (WTeeth) is a healthcare-first web platform focused on secure access, streamlined clinical workflows, and patient-friendly experiences. It provides a component-driven UI system, strong authentication, and data visualization patterns to accelerate delivery of modern healthcare products.

## Highlights

- **Reusable UI system** for consistent, accessible clinical interfaces.
- **Authentication and access control** with secure sessions and protected routes.
- **Data visualization** for outcomes, KPIs, and care tracking.
- **Extensible architecture** designed to scale with integrations and new modules.

## Tech Stack

- **Next.js**
- **React**
- **TypeScript**
- **Tailwind CSS**
- **Prisma**
- **NextAuth (Auth.js)**
- **Zod**
- **date-fns**

## Ideal For

- Healthcare startups validating product ideas quickly.
- Clinics needing a polished, secure patient experience.
- Recruiters assessing healthcare-focused full-stack work.
