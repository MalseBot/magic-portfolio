## Overview

A modern social app with a stunning UI with a native mobile feel, a special tech stack, an infinite scroll feature, and amazing performance using React JS, Appwrite, TypeScript, and more.

## Key Features

- **User authentication** Allows users to create accounts and log in securely using Zod for vaildation.
- **Post creation** Enables users to create and share posts with images and captions.
- **Comment threading** Allows users to comment on posts and view replies in a threaded format.
- **Search functionality** Allows users to search for posts and profiles based on keywords.
- **Infinite scroll** Enables users to load more posts and comments as they scroll.

## Technologies Used

- **Appwrite** A backend-as-a-service platform that provides a range of features, including authentication, storage, and database.
- **TypeScript** A statically typed superset of JavaScript that adds optional types to the language.
- **NEXT JS** A JavaScript library for building user interfaces.
- **React Query** A library for managing and caching asynchronous data in React.

## Outcome

ensuring seamless cross-device compatibility and resulting in a 30% increase in user engagement.
